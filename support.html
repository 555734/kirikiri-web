Frequently Asked Questions

Can't connect to Google Cloud Shell

  • Make sure you are signed in with a Google account that has Cloud Shell enabled.
  • Check your internet connection.
  • Try signing out and signing back in from the Cloud Shell tab.
  • Cloud Shell may take up to 30 seconds on first launch.

Google sign-in fails or loops

  • Ensure you have a stable internet connection.
  • If you use 2-Step Verification, complete the verification on the same device.
  • Try signing out from the Account tab, then sign in again.

SSH connection refused

  • Double-check the hostname, port, username, and authentication type in the SSH server settings.
  • Make sure the server allows incoming SSH connections (port 22 or your custom port).
  • If using key authentication, verify the correct private key is stored for that server.

GitHub repositories not loading

  • Go to the Account tab and reconnect your GitHub account with a valid Personal Access Token.
  • The token needs repo scope at minimum.

App crashes or behaves unexpectedly

  • Force-quit and relaunch the app.
  • If the issue persists, please report it via GitHub Issues (link below).
