kirikiri プライバシーポリシー

運営者は、本アプリにおけるユーザーの個人情報の取り扱いについて、以下の通り定めます。

1. 取得する情報

本アプリでは、以下の情報を取得・使用します。

  • 認証情報: Google OAuth 認証トークン、GitHub アクセストークン、SSH接続用の秘密鍵。
  • 接続情報: 接続先のホスト名、ポート番号、ユーザー名。

2. 利用目的

取得した情報は、以下の目的でのみ使用されます。

  • ユーザーの指示に基づく Google Cloud Shell への接続および操作。
  • ユーザーの指示に基づく GitHub API を通じたリポジトリ情報の取得および編集。
  • SSH プロトコルを使用したリモートサーバーへの接続。

3. データの保存と外部送信の禁止

  • ローカル保存: すべての認証情報および設定データは、ユーザーの端末内の安全な領域（Secure Storage）にのみ保存されます。
  • 外部送信の禁止: 運営者は、ユーザーの認証情報や個人データを、運営者の管理するサーバーや第三者のサーバーへ送信・収集することはありません。通信は本アプリから各サービス（Google/GitHub）またはユーザー指定のサーバーとの間で直接行われます。

4. お問い合わせ先

本アプリに関するお問い合わせやプライバシーに関するご質問は、以下の窓口までお願いいたします。

  • GitHub: kirikiriのリポジトリ

5. 改定

運営者は、必要に応じて本ポリシーを改定することがあります。改定した場合は、アプリ内または適切な方法で通知します。
